# ARCkmerHMM
Hidden Markov Model using a negative binomial distribution and the ARCkmerFinder output to infer archaic human introgressed tracts in modern human long read assemblies. This model is based on HMMix [Skov et al., 2018](https://journals.plos.org/plosgenetics/article?id=10.1371/journal.pgen.1007641).
